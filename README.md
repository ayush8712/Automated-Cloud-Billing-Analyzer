✔ Overview
A serverless tool that automatically collects cloud billing data from AWS Cost Explorer, analyzes spending patterns, detects cost anomalies and idle resources, and sends optimization recommendations via email + dashboard.

✔ Tech Stack
AWS Lambda
AWS Cost Explorer API
Amazon S3
Amazon Athena / QuickSight
SES (Email)
Python

✔ Features
Automated bill extraction (daily/weekly/monthly)
Cost anomaly detection
Idle resource identification
Forecast next month billing
Email cost summary and recommendations
Dashboard visualization
